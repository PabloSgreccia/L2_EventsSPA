¿Qué es Sharvent?

Sharvent es una plataforma digital que te permite crear, compartir y visualizar eventos propios y de otros usuario de todo el país.


Tecnologías utilizadas:

  • HTML
  • CSS
  • Angular
  • Typescript
  • Javascript
  • NodeJs (Express y JWT)
  • Postman
  • Bases de datos SQL
  • Sequelize (ORM)
  • Versionado con Git
  • Swagger
  • Buenas prácticas
  • Trabajo en equipo
